## Quick Build Guide

### Step 1: Install Dependencies
```bash
# Ubuntu/Debian
sudo apt-get install build-essential cmake g++ libntl-dev

# macOS
brew install cmake ntl

# fedora
sudo dnf install cmake gcc-c++ ntl-devel
```

### Step 2: Install emp-toolkit

Follow the instructions in [README.md](README.md#dependencies) to install emp-tool and emp-zk.

### Step 3: Build ZKUNSAT

```bash
# Option A: Using setup script (recommended)
chmod +x setup.sh
./setup.sh

# Option B: Manual build
cmake . -DEMP_TOOL_DIR=/path/to/emp-tool -DEMP_ZK_DIR=/path/to/emp-zk
make

# Expected result: [100%] Built target test
```

### Step 4: Run the Program

```bash
# Terminal 1 (ALICE - Party 1)
./test 1 12345 127.0.0.1 /path/to/proof.prf.unfold

# Terminal 2 (BOB - Party 2)  
./test 2 12345 127.0.0.1 /path/to/proof.prf.unfold
```

## Support & Troubleshooting

### Common Issues

**"emp-zk/emp-zk.h: No such file or directory"**
- Fix: Install emp-zk or specify path with `-DEMP_ZK_DIR`
- See [README.md](README.md#troubleshooting)

**"cmake command not found"**
- Fix: Install cmake
  - Ubuntu: `sudo apt-get install cmake`
  - macOS: `brew install cmake`

**Build hangs or fails during linking**
- Fix: Verify emp-tool and emp-zk are properly compiled
- See [README.md](README.md#compilation-errors)

For more issues, see [README.md](README.md#troubleshooting)
